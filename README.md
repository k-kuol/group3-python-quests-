Group 3 Python Quests

A collection of Python programming exercises covering fundamental concepts including variables, data types, user input, conditionals, loops, functions, and interactive programs.
Overview

This repository contains 27/30 Python quest files organized across 6 levels, designed to practice core programming concepts through hands-on, adventure-themed exercises.
Quest List

Level 1: First Steps in the Arcane Arts (Basics)

Quest 1 - Your First Spell
Introduction to print(). Write a script that prints a cool welcome message for a fellow adventurer.

Quest 2 - The Naming Ceremony
Working with variables. Create variables for a hero's name and superpower, then print an introduction sentence.

Quest 3 - The Treasure Chest
Understanding data types (strings, integers, floats). Create variables for your age and favorite food, and print them in a sentence.

Quest 4 - The Town Crier
Combining variables and strings for output using concatenation and f-strings. Print a dynamic welcome message with city name, year, and your name.

Level 2: A Dialogue with the Machine (User Input & Basic Math)

Quest 6 - The Fortune Teller
Using input() to make programs interactive. Ask the user for their name and quest, then print a confirmation.

Quest 7 - The Magic Number Converter
Type conversion with int(). Ask for a birth year, convert it, and calculate the user's approximate age.

Quest 8 - The Potion Brewer
Basic arithmetic operators (+, -, *, /). Calculate the total cost of dragon scales and elf tears.

Quest 9 - The Greedy Goblin
The modulo operator (%). Distribute 27 gold pieces among 4 friends and calculate the remainder.

Level 3: The Crossroads of Logic (Conditional Statements)

Quest 11 - The Guardian of the Bridge
The if statement. Ask for the user's age and check if they are old enough to vote.

Quest 12 - The Two-Path Cave
The if-else statement. Build a password checker that grants or denies access.

Quest 13 - The Maze of Many Choices
The if-elif-else statement. Write a grading program that converts scores (0–100) into letter grades.

Quest 14 - The Logical Gatekeeper
Logical operators (and, or, not). Check if a user meets multiple conditions to enter a club.

Quest 15 - The Nested Riddle
Nested if statements. Create a mini-adventure with branching decisions inside decisions.

Level 4: The Power of Repetition (Loops)

Quest 16 - The Marching Ants
The for loop. Print numbers from 1 to 10 using a loop.

Quest 17 - The Endless Staircase
The while loop. Use a counter that increments until it reaches 5.

Quest 18 - The Loop of Riddles
while loop with user input. Build a guessing game that loops until the user guesses correctly.

Quest 19 - The Countdown
The range(start, stop, step) function. Simulate a rocket launch countdown from 10 to 1, then print "Blastoff!".

Level 5: The Alchemist's Lab (Functions)

Quest 21 - The Reusable Incantation
Defining functions with def. Create a greet_adventurer() function and call it three times.

Quest 22 - The Personalized Scroll
Functions with parameters. Create a personalized_greeting(name, quest) function using user input.

Quest 23 - The Oracle's Vision
Functions with return values. Write calculate_area(length, width) that returns the area of a rectangle.

Quest 24 - The Master Spell
Calling functions from within other functions. Build ask_for_age() and can_they_vote(age) that work together.

Level 6: The Grand Challenge

Quest 25 - The Number Wizard
Upgraded number guessing game with "too high" / "too low" feedback using a while loop.

Quest 26 - The Simple Calculator
A calculator with four operations using separate functions and an if-elif-else chain.

Quest 27 - The FizzBuzz Test
Classic FizzBuzz challenge — print numbers 1–100 with special rules for multiples of 3 and 5.

Quest 28 - The Adventure Begins
A text-based "Choose Your Own Adventure" game with functions for locations and multiple endings.

Quest 29 - The Code Breaker
Limited-attempt guessing game (3 tries to guess the secret code 42) using while loops with break statements.

Quest 30 - The Reflective Scribe
Commented versions of three previous quests, explaining what each part of the code does and why.
Running the Quests
Each quest can be run independently:

bash# Quests 1–9 have spaces in filenames
python "quest 1.py"
python "quest 2.py"

Quests 11+ have no spaces

python quest11.py
python quest12.py
